# Levantar Consults Website

Premium legal consultancy website built with React, TypeScript, and Tailwind CSS.

## Live Site
https://levantarconsults.com

## Features
- Responsive design
- Smooth scroll animations
- Contact form
- SEO optimized

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment
This site is deployed on Netlify with automatic deployments from GitHub.
